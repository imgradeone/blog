作为一名 Android 和 iOS 都使用过的博主，我体验过一些移动端文本编辑器，不得不说，市场上不错的编辑器真的太多了，我都数不过来了。😂

在长期的使用后，我选出了两款用起来很 awesome，但都是单平台独占的文本编辑器——

纯纯写作（Android） 和 Bear 熊掌记（iOS）。

纯纯写作 - 主打文章不丢失

从它的 slogan “快速、优秀体验、永不丢失内容”可以看出，作者在文本保存真的是下了大手。

而事实是真的如此。drakeet 大大在文本备份方面下足了心，毕竟他也体验过那种好不容易码了半天字，结果最后全丢的那种感觉。 换作我，就直接当场去世了 😂

先不提文本内容保护。先来说说写作体验吧。

舒适的写作体验

打开 app，第一眼看见的就是写作界面。你的灵感，第一时间就可以尽情挥洒。当然，作者大大的感谢信也很有意思哦。❤️

整个界面只有你的内容，不得不说，这才是最舒服的写作方式。对于暗黑色的背景，不满意也可以更换，稿纸列表里还有 drakeet 在 Unsplash 精选的一些背景图，用起来也很 exciting。当然，作为 Pro 用户（$），你还可以自定义背景色和图片，让写作成为享受。

有没有注意到底下的快捷输入栏？在这里，你可以将一些自己常用的词语放进这里，对于写小说的人来说，不用去回忆“这是谁来着”，然后再手输角色名了；对于一些需要经常复制粘贴的人来说，也十分有用~~；对于部分喜欢狂按的人来说，emmm，你加油~~。

如果你想专注于写作，也可以试一下“状态栏专注模式”，平时花花绿绿的状态栏突然安静，又是另一种舒适。（虽然有点 bug，部分设备可能无法正常使用）

Markdown = More

由于纯纯写作兼容 Markdown 语法，因此你可以借助它来插入图片、绘制表格等，几乎 Markdown 所有能做到的，它也可以做到。

在快捷输入栏把“MD”的框勾选上就可以进入 Markdown 模式了。

如果不知道 Markdown 怎么用，或者忘记语法了，小键盘的各种 Markdown 工具也会帮助你进行编辑、排版的。

虽然在加入语法后没有相关高亮，也没有“所见即所得”的快感（毕竟搞了的话，纯纯写作杀马特化不可避），但是有了舒适的体验与足量的个性化功能，没有这种杀马特 UI 也没关系。

舒心的文章保护机制

前文里提到了纯纯写作的备份功能，实际上，它不止于一个功能。

它是一整套文章保护机制。

毫不夸张地说，在你每打一个字的时候，纯纯写作就已经开始主动保存你的写作内容。一旦保存失败，它将立马提示用户，并且尽可能让你保存文章，保证数据安全。

同时，就算再手残，不小心卸载了软件，也可以恢复备份，只需一键恢复，所有文章、设置，甚至包括纯纯写作 Pro 版的激活码，都瞬间恢复了。配合云备份功能，除非 WebDAV 跑路或者进不去 Dropbox（取决于设定），你可以随时保存你的备份，换机也无忧了。

0 延迟的同步写作（$）

虽然这是 Pro 版的功能，且十分初级，能不能用还有点“随缘”（实际上并没有），但是，

你有没有体验过手机无线连接电脑，写作还毫无延迟的那种感觉吗？

这就是纯纯写作的桌面版，支持 Windows 和 macOS。（本来还支持 Linux 的，但 drakeet 没编译😶坐等 AUR 大佬）

打开界面，只有一些提示文字，真的十分简洁，但也有点朴素了。

按照提示连接手机，手机上打开的文章就自动传送到电脑上了。接下来，电脑的键盘就成为你手机上纯纯写作的键盘了。你每打一个字，手机上也会立马反应，反之，在手机上打字，电脑也会立马反应，几乎没有延迟。虽然目前的 UI 过于朴素，以及 emoji 暂时无法在电脑端展示，但不影响它在 Android 端的口碑。

我把桌面版纯纯写作当成像 iOS “接力”（Handoff）功能一样的存在，因为我可以做到没空开电脑时开电脑时使用手机写作，使用桌面版进行文章编辑，完成后再用桌面版或者有一点点难用（但实际体验还行）的云读写进行最终的文章发布。

其他细节功能

对于写作功能方面，纯纯写作丝毫不逊色。在快捷输入栏的小键盘区域，提供了许多格式化功能，排版也更为方便。其中“在中英文之间插入空格”功能对 少数派 等网站的排版起到节约时间的作用。

在篇章管理列表（侧滑栏）中，你可以选择只显示标题，也可以显示摘要；如果你有许多篇文章，或者你的小说分章/专栏分组，也可以创建一个卷，进行文章的归类，同时也可以对不需要使用的卷折叠起来，看上去就很清爽。

如果你是小说作者，当你完成整本书的写作后，可以使用“合并成书导出”（$）功能，将你的所有章节以一个 txt 文本文件导出，接下来可以使用你喜欢的小说软件导入并阅读了。不过似乎作者还没意识到 epub 格式诶

此外，借助“预览 | 另存为…”（）功能，可以将文章导出为 txt/md/长图片；借助“查找 & 替换”（?）功能，可以拥有类似于 Visual Studio Code 的 Ctrl + F 功能。即便在小部分场景下，“查找”功能会有一些 bug，导致查找和替换时结果不完整，其他的还是十分优秀。

更多的细节优化，还是需要在体验中慢慢感受的。

纯纯写作 - 总结

纯纯写作是我在 Android 端体验的最佳编辑器。虽然对于部分场景来说还不是最佳的，但是在总体上，它几乎无人能敌。不丢失文本，简约的 UI，高度的个性化功能，使其成为一款优秀的 app。
